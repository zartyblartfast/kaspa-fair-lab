# Verifier

This folder is intentionally minimal for the initial scaffold.

## Purpose

Store lightweight methods and notes used to verify Toccata covenant claims.

## Current scope

- Command transcripts.
- Checklists for what can be independently repeated.
- Notes on assumptions and evidence gaps.

## Not yet implemented

- No automated verifier pipeline yet.
- No proof artifact generator yet.
- No full audit tooling yet.

## Planned future content

1. Reproducibility checklist.
2. Evidence bundle format (hashes, txids, and context).
3. Explicit trust assumptions.
